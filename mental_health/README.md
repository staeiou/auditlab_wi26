## Mental Health Triage Audit
This folder contains documents relating to the data analysis and statistical testing of the mental health audit. The prompt and variables used can be found in the
Introduction section of analysis.ipynb. 

### Structure
mental_health/
├─ analysis.ipynb           # Jupyter Notebook with analysis, visualizations & statistical testing
├─ medical_data.csv         # Dataset obtained from mental health audit
├─ requirements.txt         # dependencies 

In order to analyze and run analysis.ipynb, begin by running the requirements.txt to install the necessary Python package dependencies:
```
pip install -r requirements.txt
```
