## Legal Services Audit
This folder contains documents relating to the data analysis and statistical testing of the legal services audit.

### Structure
```
legal_service_audit/
├─ analysis.ipynb           # Jupyter Notebook with analysis, visualizations & statistical testing
├─ results.csv              # Dataset obtained from employment screening audit
├─ requirements.txt         # dependencies 
```

In order to analyze and run analysis.ipynb, begin by running the requirements.txt to install the necessary Python package dependencies:
```
pip install -r requirements.txt
```
